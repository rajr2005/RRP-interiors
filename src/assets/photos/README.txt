Add your gallery images in these folders:

- src/assets/photos/bedroom
- src/assets/photos/hall
- src/assets/photos/kitchen

Supported file types:
- .jpg
- .jpeg
- .png
- .webp
- .avif
- .gif

Example:
- src/assets/photos/bedroom/bedroom-1.jpg
- src/assets/photos/hall/hall-design.png
- src/assets/photos/kitchen/modular-kitchen.webp

When you add images to a folder, they will appear automatically in the Photos page under that category.
